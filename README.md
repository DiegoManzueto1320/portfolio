# Portfolio — Diego Manzueto

Site web personnel présentant des projets et compétences (Data Analyst / Développeur).

Installation locale

- Installer WAMP/XAMPP et placer le dossier `Portofolio` dans le répertoire web.
- Ouvrir http://localhost/Portofolio/index.php

Script Python (optionnel)

- Activer l'environnement virtuel : `.venv\Scripts\Activate.ps1`
- Lancer le script : `python optimize_images.py`

Remarques

- Le dossier `.venv/` et `data/contacts.csv` sont ignorés par `.gitignore`.
