 # DashbordGestionProjet
Ce dépôt présente un tableau de bord structuré et harmonisé dédié à la gestion de projets informatiques. Il regroupe l’ensemble des éléments essentiels permettant le suivi et l’analyse des activités. Le tableau de bord a été conçu et réalisé avec Power BI, offrant une visualisation claire et interactive des données.
